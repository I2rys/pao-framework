# Pao Framework
Pao framework.

## Installation
Github:
```
git clone https://github.com/I2rys/pao-framework
```

NpmJS:
```
npm i readline-sync request-async
```

## Setup
If somehow you want to use Pao for production just change the **Pao > url** with your hosted Pao server URL.

## Usage
```
node index.js
```
